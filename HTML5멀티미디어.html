HTML5 멀티미디어

video

controls : 플레이 버튼과 남은시간, 소리 표시
autoplay : 자동실행 (단, muted상태일때만 자동실행됨)
muted : 무음
loop : 반복재생

플러그인

object


object태그는 타양한 파일들을 화면에 출력하는 태그이다

embed


embed태그는 object와 동일하게 사용할 수 있다
embed태그를 사용하는게 좋다

유튜브 영상 삽입하기

소스코드로 복사해오면 iframe안에 설정이 되어서 들고올 수 있다

video의 설정을 유튜브에서도 적용할 수 있다
url뒤에 ?를 이용한 쿼리스트링으로 작성할 수 있다
?mute=1&autoplay=1 같은 값으로 들어간다
1은 true 참이라는 값으로 사용, 0은 false 거짓이라는 값으로 사용된다
