input요소

datalist는 input의 값중에 몇개를 리스트로 띄워주는 역할을 한다
input태그와 datalist태그는 list와 id로 연결이 되어있다

keygen은 지금은 지원하지 않는 태그이다
웹의 발전에 따라서 태그들이 생성되거나 쓰이지 않는다

+ = 0

output태그는 form의 oniput 속성과 함께 사용한다
oninput에 들어간 자바스크립트의 명령어를 계산해서 쓸 수 있다
보통은 자바스크립트를 script 태그 안에서 사용한다

input 추가 타입


range 타입은 값을 슬라이더를 통해서 선택을 할 수 있다
선택한 값은 자바스크립트를 통해서 가져오거나 form을 통해서 값을 전달할때 알수있다







email의 경우는 이메일의 형태를 갖추고 있는지 유효성 검사를 해준다 (validation)

form에 novalidate속성이 들어가게 되면 유효성 검사를 하지않는다



input요소의 속성

required는 값이 없을때 서버로 보내는 것을 막는 속성이다
